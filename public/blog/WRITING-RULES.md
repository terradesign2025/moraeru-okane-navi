# もらえるお金ナビ ブログ記事執筆ルール v1.0
最終更新: 2026-05-14

---

## 🎯 コンセプト
「知っている人だけが得をする」お金の情報を、週刊誌・YouTube サムネ風に面白くわかりやすく届ける。
難しい知識ゼロ。ママ・パパ・シニア・移住検討者など誰でも読める。

---

## 📌 タイトルルール（毎記事必須）

| ルール | 例 |
|--------|-----|
| 驚き数字を入れる | 「最大106万円」「全額7万3千円」 |
| 感情ワードを入れる | 「神対応」「ヤバい」「衝撃」「太っ腹」 |
| 「!?」「！」を使う | 「えっ、これでもお金もらえるの!?」 |
| 地名を入れると具体的 | 「明石市」「御代田町」「岡崎市」 |
| 週刊誌風フォーマット | 「【罠】○○で0円」「【衝撃】○○が無料」 |

---

## 🏗 記事構造（必ず守る）

```
1. meta title / description（SEOキーワード含む）
2. カテゴリタグ + 大見出し（h1）
3. 記事メタ（日付・タグ・読了時間）
4. リード文（3行以内・読者への問いかけ→驚き事実の提示）
5. h2①：この制度、何がもらえる？（概要）
6. h2②：あなたは対象？（条件・要件）
7. h2③：具体的にいくら？（金額・シミュレーション）
8. ↑ここにアプリバナー（中間）
9. h2④：申請方法（ステップ形式）
10. h2⑤：よくある失敗・注意点 OR Q&A
11. ↑ここにアプリバナー（末尾）
12. 関連記事4本
13. フッター
```

---

## ✍️ 文体ルール

- **語尾**: です・ます体。柔らかく「〜ですよね」「〜しませんか？」
- **ツッコミ**: 「こんな制度、信じられますか？」「驚いてしまいますよね」
- **つなぎ**: 「実は…」「驚くことに」「太っ腹すぎる」「見逃すと損！」
- **読者に語りかける**: 「あなたの家庭はどうでしょうか？」「今すぐ確認してみてください」
- **難しい言葉は禁止**: 「補助額の上限設定」→「もらえる上限」に言い換える

---

## 🎨 視覚要素（1記事に最低2つ使う）

| 要素 | 使う場面 |
|------|---------|
| ハイライトボックス（緑） | 主要な金額・まとめポイント |
| 警告ボックス（黄） | 申請期限・注意事項 |
| 情報ボックス（青） | 豆知識・補足情報 |
| 比較表 | 複数市区の比較・金額一覧 |
| ステップボックス（黒） | 申請手順（5ステップ形式） |
| ランキングカード | TOP形式・複数都市比較 |
| シミュレーション計算 | 「こうすると合計○○万円」 |

---

## 🔗 内部リンクルール

- **アプリバナー**: 記事中間・末尾の2箇所（必ず2箇所。絶対省略しない）
- **関連記事**: 末尾に必ず4本（クロスリンク）
- **注意書き**: 末尾フッター前に「公式サイトで確認を」ボックス（必須）

---

## 📊 SEOルール

- h1にメインキーワード（「○○補助金」「もらえるお金」など）を含める
- meta descriptionは100-120文字でキーワード2-3個自然に含める
- 記事内でメインキーワードを3-5回自然に繰り返す
- 画像代わりに絵文字を活用（サムネイル的役割）

---

## 📏 文字数

- **最小**: 1,500文字（短すぎはSEO的にNG）
- **推奨**: 1,800〜2,200文字
- **最大**: 2,500文字（長すぎると離脱率UP）

---

## 📁 ファイル命名規則

```

例: 008-akashi-omutu.html
    009-nenkin-midori.html
    010-eakon-mitodamachi.html
```
